geoscape
	universe 
		zarzadza wszystkimi swiatami 
	world 
		2D map of world tiles, many worlds, earth is 90x45, 500km = 1 tile
		zawiera w sobie liste prowincji 
	world tile 
		single tile on a world, has a simple terrain type (water, land, land rough, land very rough) to control cost of travel 
		not used for game play other then world path 
	world path 
		List of tiles betwen 2 provinces with shortest path for a craft of specific type (water, land, air etc). Used to calculate range of craft for travel 
	province 
		main entity in which game exist, like a graph node, can contain single player base, many enemy missions, many player crafts. Has economy power and Biome. 
	biome 
		define fauna, flora , terrain of province. Define interception background image. Define list of potential terrains for land battlescape. Define is it land / water.
	region 
		each province belong to region. 
		All score and missions are regions specific. 
		Provide some weights for missions probablity, can limit options in base (purchase only in region X)
	portal 
		mechanizm ktory pozwala na bezkosztowy ruch pomiedzy dwoma prowincjami dla Craftów, nawet jesli to provincje z innych worlds. 
	country 
		political entity that must be protected by player, has economy power based on sum of its provices. Some provinces belong to country. Player has relation with country. Country provide fuding to player every month based on score from missions. Higher score -> better relationship -> improved funding level. Relation with country defines missions and funding levels. 
	travel 
		craft is sent on a mission, he perform a move between provinces using world tile and paths. Craft speed defines how many travel can perform per turn . 
	radar coverage - 
		each craft and base with radar with perform once per day scan in nearby provinces in range to detect missions. Mission has cover / cover change stat and once this stat is above zero then mission is detected and can be intercepted. 
	
	overview
		- build new base 
		- generate missions in provinces (covered)
		- use bases or crafts to detect missions with radar coverage 
		- sent craft from base to province with mission using travel / world path mechanics 
		- start interception 
		- interception is either succesful or not, depends on it either return to geoscape or move to battlescape with units 
		- all missions MUST have interception thou sometimes it can be very short 
	geoscape wyglada jak gra eador, heroes of might magic, gdzie craft = hero a units = jego armia i poruszamy sie po okreslonym swiecie
